# Carpeta de Imágenes

Esta carpeta contiene todas las imágenes del sitio web de la boda.

## Estructura

- `hero-bg.jpg` - Imagen de fondo del Hero Section
- `clouds.png` - Imagen decorativa de nubes
- `gallery/` - Galería de fotos de la pareja
  - `photo-1.jpg`
  - `photo-2.jpg`
  - `photo-3.jpg`
  - `photo-4.jpg`
  - `photo-5.jpg`

## Notas

- Las imágenes deben estar optimizadas (máximo 500KB cada una)
- Usar formato WebP cuando sea posible para mejor compresión
- Mantener resolución mínima de 1920x1080 para imágenes de fondo
